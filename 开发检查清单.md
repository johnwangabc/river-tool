# 开发检查清单

## 环境准备

- [ ] 已安装 Node.js 16+
- [ ] 已安装 npm 或 yarn
- [ ] 网络连接正常
- [ ] (可选) 已注册 Expo 账号

## 项目设置

- [ ] 运行 `install.bat` 或 `npm install` 安装依赖
- [ ] 依赖安装成功，无错误
- [ ] 手机已安装 Expo Go 应用

## 本地开发测试

- [ ] 运行 `start.bat` 或 `npm start` 启动服务器
- [ ] 服务器启动成功，显示二维码
- [ ] 手机扫描二维码，应用正常加载
- [ ] 测试各个功能页面是否正常显示
