## 巡河宝统计工具使用说明（用户版）

应用由两个底部标签组成：**综合统计** 和 **设置**。先在设置页完成令牌配置，再进行统计和导出。

### 1. 首次使用
1) 打开应用，先进入底部的 **设置** 页。  
2) 在 **认证令牌 (Token)** 输入完整的 Authorization 值，需以 `Bearer ` 开头；右侧眼睛可切换显示/隐藏。  
3) **组织 ID** 默认为 `843`，如后台组织不同请修改。  
4) 点击 **保存设置**。如需重置，点击 **清除设置**。  
5) 状态提示会显示是否已填写令牌长度；令牌和组织 ID 仅保存在本地设备。

> 获取 Token： 问开发软件的人要。

### 2. 进行综合统计
1) 确保已保存正确的 Token 和组织 ID。  
2) 在 **综合统计** 页输入 **开始日期**（格式 `yyyy-MM-dd`，如 `2024-01-01`），统计该日及之后的数据。  
3) 点击 **开始统计**，等待进度提示（会显示巡护抓取、评测抓取、活动处理、综合计算等阶段）。  
4) 完成后会显示摘要 Chips 和前 20 行表格，包含巡护/评测/活动次数及总计。  
5) 如需完整数据，点击 **导出 Excel**；会生成文件并通过系统分享保存或发送。

### 3. 导出与查看
- 导出按钮仅在统计完成后出现，文件名会包含日期。  
- 生成后弹出系统分享，可存本地、发邮箱或发送给协作方。  
- 前 20 行在应用内可直接查看，更多数据请通过导出的 Excel 查看。

### 4. 常见问题
- **提示“请输入认证令牌”**：先到设置页填 Token 并保存。  
- **数据为空**：检查开始日期格式是否正确；确认 Token、组织 ID 与后台环境一致；确保网络可用。  
- **401/权限错误**：Token 可能过期或环境不匹配，重新获取并保存。  
- **导出失败或无法分享**：确认设备存储/分享权限正常，重试导出。

### 5. 安全说明
- 认证令牌与组织 ID 仅存储在本地 AsyncStorage，不会上传服务器。  
- 导出文件可能包含个人数据，分享前请按需脱敏或限制范围。
