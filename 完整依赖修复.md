# 依赖问题修复记录

## 遇到的错误及解决方案

### 错误 1: expo-router
```
This is the first page of your app.
```
**原因**: 错误配置了 expo-router
**解决**: 移除 expo-router，改用传统 React Navigation

### 错误 2: expo-asset
```
Error: The required package `expo-asset` cannot be found
```
**原因**: Expo 52 需要 expo-asset
**解决**: `npm install expo-asset`

### 错误 3: expo-font
```
Unable to resolve "expo-font" from "node_modules\@expo\vector-icons\build\createIconSet.js"
```
**原因**: vector-icons 需要 expo-font
**解决**: `npm install expo-font`

## 如果遇到其他缺失依赖

运行自动修复：
```bash
npx expo install --fix
```

或手动安装常见依赖：
```bash
npm install expo-font expo-asset expo-constants expo-modules-core
```

## 完整重置步骤

如果问题持续，执行完全重置：

```bash
# 1. 删除所有
rm -rf node_modules package-lock.json .expo

# 2. 重新安装
npm install

# 3. 安装可能缺失的 Expo 包
npm install expo-font expo-asset

# 4. 清理启动
npx expo start -c
```
