     河流数据统计工具 - 快速打包指南
═══════════════════════════════════════════════════════════════

📦 一键打包
──────────────────────────────────────────────────

最简单的方式：

    python build_exe.py

打包脚本会自动完成所有步骤！


📋 手动打包（如果自动脚本失败）
──────────────────────────────────────────────────

1. 备份并替换配置文件
   copy config.py config.py.backup
   copy config_exe.py config.py

2. 安装打包工具
   pip install pyinstaller

3. 执行打包
   pyinstaller 巡河宝统计工具.spec

4. 恢复配置
   move config.py.backup config.py

5. 复制配置文件到输出目录
   copy user_config.ini dist\user_config.ini


📂 打包后的文件
──────────────────────────────────────────────────

dist/
├── 巡河宝统计工具.exe  ← 主程序
└── user_config.ini      ← 配置文件（只需修改token）


🚀 使用方法
──────────────────────────────────────────────────

1. 打开 dist 文件夹
2. 编辑 user_config.ini，填入正确的token：

   [API]
   auth_token = Bearer YOUR_TOKEN_HERE

3. 双击运行 巡河宝统计工具.exe


🔒 安全特性
──────────────────────────────────────────────────

✓ 只有token需要配置，其他参数已内置
✓ 用户看不到API地址、组织ID等敏感配置
✓ token可以随时修改，无需重新打包


⚠️ 注意事项
──────────────────────────────────────────────────

• 确保user_config.ini与exe在同一目录
• token格式：Bearer eyJ0eXAi...（完整token）
• 首次运行会创建logs目录存放日志
• 生成的Excel文件默认保存在桌面


📞 故障排除
──────────────────────────────────────────────────

问题：打包失败
解决：删除build和dist文件夹后重试

问题：exe无法运行
解决：查看logs目录中的日志文件

问题：提示token未配置
解决：检查user_config.ini文件内容和位置
