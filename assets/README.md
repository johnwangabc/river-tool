# Assets 目录

此目录存放应用图标和启动画面资源。

## 需要的资源文件

在使用 Expo 云打包之前，请准备以下图片资源：

### 1. icon.png
- **尺寸**: 1024x1024 像素
- **格式**: PNG (透明背景)
- **用途**: 应用图标
- **说明**: 这是应用的主图标，会自动调整为各平台所需尺寸

### 2. adaptive-icon.png (Android)
- **尺寸**: 1024x1024 像素
- **格式**: PNG (透明背景)
- **用途**: Android 自适应图标
- **说明**: 前景图标，背景色在 app.json 中配置

### 3. splash.png
- **尺寸**: 1242x2436 像素 (或更大)
- **格式**: PNG
- **用途**: 启动画面
- **说明**: 应用启动时显示的画面

### 4. favicon.png (Web)
- **尺寸**: 48x48 像素
- **格式**: PNG
- **用途**: Web版本图标

## 快速生成

如果没有设计资源，可以使用以下方法：

### 1. 使用在线工具
- 访问 https://www.figma.com/
- 创建 1024x1024 的画布
- 设计简单的图标
- 导出为PNG

### 2. 使用Expo默认图标
保持现有的占位符图标即可，但建议最终替换为自定义图标。

### 3. 使用图标生成器
- https://icon.kitchen/ (免费在线图标生成)
- https://www.canva.com/ (在线设计工具)

## 图标设计建议

1. **简洁明了**: 使用简单的图形和图标
2. **颜色鲜明**: 使用高对比度的颜色组合
3. **主题相关**: 可以使用水滴、河流、统计图表等元素
4. **留白充足**: 确保图标周围有适当的留白

## 替换步骤

1. 准备好上述尺寸的图片文件
2. 将文件放入此 `assets/` 目录
3. 确保文件名正确
4. 运行 `eas build` 即可

## 当前状态

目前项目使用 Expo 默认占位符图标。建议在正式发布前替换为自定义图标。
